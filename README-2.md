## 🌍 Dataset

**Source:** World Bank Open Data API (live fetch) / WHO African Health Indicators  
**Year:** 2021  
**Coverage:** 40 African countries across 5 regions  

### Indicators
| Column | Description | Source |
|--------|-------------|--------|
| `life_expectancy` | Life expectancy at birth (years) | World Bank SP.DYN.LE00.IN |
| `under5_mortality` | Under-5 mortality rate (per 1,000 live births) | World Bank SH.DYN.MORT |
| `gdp_per_capita` | GDP per capita (current USD) | World Bank NY.GDP.PCAP.CD |
| `electricity_access_pct` | Access to electricity (% of population) | World Bank EG.ELC.ACCS.ZS |
| `literacy_rate_pct` | Adult literacy rate (%) | World Bank SE.ADT.LITR.ZS |

## 📓 Notebook Contents

### Step 1: Download Data
- Fetches real data from **World Bank API** using `requests`
- Falls back to a realistic synthetic dataset (based on published WHO/WB figures) if API unavailable

### Step 2: Explore with Pandas
```python
df.shape       # (40, 9)
df.info()      # data types and non-null counts
df.describe()  # statistical summary
```

### Step 3: Data Cleaning
- ✅ Removed duplicate rows
- ✅ Fixed data types (numeric columns, category for region)
- ✅ Filled missing values with **regional medians** (more accurate than global mean)
- ✅ Capped outliers using IQR method
- ✅ Added `income_group` classification (World Bank thresholds)

### Step 4: Five Visualisations

| Plot | Type | What it shows |
|------|------|---------------|
| 1 | **Histogram** | Distribution of life expectancy with KDE, mean & median |
| 2 | **Scatter Plot** | GDP per capita vs life expectancy by region with trend line |
| 3 | **Bar Chart** | Average life expectancy & electricity access by region |
| 4 | **Box + Violin Plot** | Under-5 mortality and GDP distribution by income group |
| 5 | **Correlation Heatmap** | Pearson correlations between all 5 indicators |

### Step 5: Key Insights
- North Africa has the highest average life expectancy (~74 years)
- Strong positive correlation between GDP and life expectancy (r ≈ 0.75)
- Electricity access and literacy rate are the strongest predictors of health outcomes
- Low-income countries have 4–5× higher under-5 mortality than upper-middle income
